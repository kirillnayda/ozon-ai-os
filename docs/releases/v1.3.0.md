# Ozon AI OS v1.3.0

- Добавлен выбор товара в Telegram-разделе «Остатки по кластерам».
- Карточка товара показывает доступный остаток и средние продажи по каждому кластеру, а также итоги за день.
- Добавлены пагинация товаров и кнопки возврата в главное меню.
- Добавлена безопасная команда `/ozon_contracts`, которая получает read-only ответы Ozon API, обезличивает их и отправляет contract fixture владельцу в Telegram.
- Production-import остаётся fail-closed до проверки полученного контракта и добавления contract-тестов DTO.
