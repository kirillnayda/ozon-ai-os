# Ozon AI OS v1.3.1

- Updater восстанавливает владельца и групповые права чтения исходных файлов после `git checkout`, поэтому systemd-пользователь `ozonai` продолжает запускать бот.
- То же восстановление прав выполняется при автоматическом rollback.
- `/ozon_contracts` теперь сообщает безопасный HTTP-статус неуспешного Ozon API-запроса, не сохраняя тело ответа, ключи или заголовки.
- После успешного health-check updater отправляет владельцу явное Telegram-уведомление с установленной версией.
