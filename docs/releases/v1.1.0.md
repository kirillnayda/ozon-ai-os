# Ozon AI OS v1.1.0

- Пошаговый Telegram-менеджер FBO-поставок с тестовым PDF.
- Создание черновика из рекомендаций по среднесуточному спросу.
- Идемпотентность, восстановление операций и PDF-outbox.
- История, метрики, редактирование состава и русское меню.
- FBO-остатки, кластерные отчёты, контроль устаревания и дефицита.
- Безопасное обновление через `/update` и подписанные GitHub Releases.

Production-запись в Ozon остаётся fail-closed до добавления официальных DTO fixtures и contract-тестов из реального кабинета.
