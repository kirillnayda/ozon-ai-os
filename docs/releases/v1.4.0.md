# Ozon AI OS v1.4.0

- Включён production read-only импорт остатков из подтверждённого `/v1/analytics/stocks`.
- SKU загружаются с пагинацией `/v3/product/list` и отправляются в аналитику пакетами до 100.
- Остаток берётся из `available_stock_count`, средние продажи по кластеру — из `ads_cluster`.
- Спрос не дублируется при нескольких складах одного товара в одном кластере.
- Синхронизация работает при любом `LIVE_MODE`; операций записи в Ozon нет.
