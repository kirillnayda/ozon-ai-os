# Ozon AI OS v1.2.1

Hotfix первоначальной установки на Ubuntu.

## Исправления

- `scripts/check.py` больше не зависит от текущего рабочего каталога установщика;
- пользователь systemd-сервиса `ozonai` получает доступ на чтение и выполнение к каталогу приложения и virtualenv;
- добавлены регрессионные тесты запуска проверки из стороннего каталога и прав установщика.

## Проверка

```bash
python scripts/check.py
```

Production-запись в Ozon остаётся заблокированной до подтверждения контрактов Seller API и добавления contract-тестов.
