# Product Vision Ozon AI OS

Ozon AI OS — контролируемая серверная система управления магазином через Telegram. Человек принимает решения, сервис собирает данные, рассчитывает рекомендации и автоматизирует только подтверждённые операции. Безопасность, аудит и обратимость важнее автономности.

Developer Agent — внутренний контур разработки: превращает Telegram-задачу в отдельную проверенную Git-ветку. Он не выполняет merge и production deploy и не имеет доступа к рабочим секретам.
